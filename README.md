# Microcontroller Projects | Embedded Systems Portfolio

## Professional Summary
This repository contains a structured collection of **microcontroller-based embedded systems projects** developed across **8-bit, 16-bit, and 32-bit architectures**. The projects demonstrate hands-on experience in **embedded C programming, peripheral interfacing, and hardware–software integration**, suitable for academic, training, and entry-level embedded systems roles.

## Technical Skills Demonstrated
- Embedded C Programming  
- Microcontroller Architecture & Register-Level Programming  
- Digital I/O Interfacing  
- Timers, Counters, and Interrupts  
- UART Serial Communication  
- ADC and PWM Peripheral Interfacing  
- USB Fundamentals  
- ARM-Based Microcontroller Programming  
- Hardware Debugging and Testing

### 8-bit Microcontrollers

  - GPIO interfacing
  - Timers and counters
  - UART serial communication

  - External interrupts

### 16-bit Microcontrollers

  - GPIO and interrupt handling
  - ADC interfacing
  - PWM control
  - USB communication basics

### 32-bit Microcontrollers
- **ARM7 – LPC2148**
  - GPIO, timers, and interrupts
  - UART and peripheral interfacing
  - ARM7 architecture fundamentals

- **ARM Cortex-M4 – LPC4088**
  - High-performance embedded applications
  - Advanced peripheral interfacing
  - ARM Cortex-M architecture concepts
