# Microcontroller Projects | Embedded Systems Portfolio

## Professional Summary
This repository contains a curated collection of **microcontroller-based embedded systems projects** developed using **8051** and **PIC18F4550** microcontrollers. The projects demonstrate practical skills in **embedded C programming, peripheral interfacing, and hardware–software integration**, suitable for academic work and entry-level embedded systems roles.

## Technical Skills Demonstrated
- Embedded C Programming  
- Microcontroller Architecture & Register-Level Programming  
- Digital I/O Interfacing  
- Timers, Counters, and Interrupts  
- UART Serial Communication  
- ADC and PWM Peripheral Interfacing  
- USB Fundamentals (PIC18F4550)  
- Hardware Debugging and Testing  

## Microcontrollers Covered

### 8051 Microcontroller
- GPIO control and interfacing  
- Timer and delay-based applications  
- UART serial communication  
- Display interfacing (LED, LCD, seven-segment)  
- External interrupt handling  

### PIC18F4550 Microcontroller
- GPIO and interrupt configuration  
- ADC-based sensor interfacing  
- PWM signal generation  
- USB communication basics  
- Peripheral configuration using MPLAB X
